
# PetBooking | App de reservas de turno




## Tecnologias

Vite, React, Tailwind CSS, Redux, Notistack


## Run Locally

Clone the project

```bash
  git clone https://github.com/Leon2192/petbooking
```

Go to the project directory

```bash
  cd petbooking
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
